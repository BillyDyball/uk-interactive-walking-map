# UK Interactive Walking Map

An interactive map of long-distance walking paths across the UK, built with [Leaflet](https://leafletjs.com/) and OpenStreetMap. Search and explore paths with a simple browser-based interface.

Path data sourced from the [LDWA Long Distance Paths](https://ldwa.org.uk/ldp/members/search_by_path.php).

## Getting Started

1. Clone the repo
2. Run `python3 -m http.server 8000`
3. Open `http://localhost:8000/ldwa_map.html`

No build tools or dependencies required — it runs entirely in the browser.
